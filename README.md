## 🎼 Features Overview

Omnisphere 2 offers several tools for sound design. These features help you craft unique textures.

*   Sound Library: Access over 14,000 sounds curated by sound designers. 
*   Granular Synthesis: Manipulate audio samples at the microscopic level to create textures.
*   Wavetable Synthesis: Use custom wave shapes to generate complex tones.
*   Hardware Integration: Connect your physical synthesizers to control the software interface.
*   Effects Rack: Apply processors like compressors, reverbs, and delays directly inside the plugin.
*   Arpeggiator: Sequence notes automatically with rhythmic patterns.

## 💡 Troubleshooting Common Issues

If you encounter problems, check these items first.

*   Plugin not found: Ensure the installer pointed to the correct VST folder used by your DAW. Perform a deep scan in your DAW settings.
*   Missing sounds: Confirm that the large sound library folder exists on your drive. You can point the plugin to this specific path in the Omnisphere settings menu.
*   High latency: Open your audio interface settings in your DAW. Lower the buffer size to reduce the time between hitting a key and hearing a sound.
*   Audio crackling: If you hear glitches, your computer might struggle with the processing load. Increase the buffer size in your DAW to fix this.
*   Compatibility: Always run the latest version of your DAW to ensure support for newer plugin standards.
